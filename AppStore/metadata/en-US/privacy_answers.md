# App Privacy Answers

Use these answers for App Store Connect's App Privacy section, assuming the
submitted binary remains equivalent to the current code.

Data Collection:
- The app does not collect data from this app.

Tracking:
- The app does not track users.

Linked to User:
- Not applicable.

Data Used to Track You:
- None.

Diagnostics:
- The app writes a local diagnostic log capped at about 256 KB. The log is not
  transmitted to the developer or any server.

Network Access:
- The app contacts GitHub for a public catalog and Google's public CDN for
  wallpaper images. These services may receive normal HTTP request data such
  as IP address and headers under their own policies, but EarthLens does not
  send account information, identifiers, telemetry, or usage data.
