# EarthLens App Store Submission Pack

Use this folder as the source of truth when filling out App Store Connect.

Apple's current Mac screenshot requirement is 1-10 `.png`, `.jpg`, or `.jpeg`
screenshots at one of these 16:10 sizes: 1280x800, 1440x900, 2560x1600, or
2880x1800. App previews are optional; if you create one for Mac, it must be a
15-30 second landscape video at 1920x1080.

Required URLs:

- Privacy Policy URL: host `PRIVACY_POLICY.md` publicly before submission.
- Support URL: must be a public page with real contact information.
- Marketing URL: optional, but recommended if you have a product page.

Prepared screenshots:

- `screenshots/mac/01-earthlens-main.png`
- `screenshots/mac/02-earthlens-setup.png`

Recommended App Store Connect setup:

- Platform: macOS
- Name: EarthLens
- Bundle ID: `com.jashanveer.earthlens`
- SKU: `earthlens-mac`
- Primary category: Photography
- Price: Free
- In-app purchases: None
- Age rating: 4+
- Content rights: Uses third-party Google Earth View imagery. Submit only if
  you are comfortable representing that the app has the necessary rights or
  permitted use for all territories where it is available.
