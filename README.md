# EarthLens

> Earth, on your desktop.

A native macOS menu-bar app that sets Google Earth View satellite imagery as your wallpaper, on demand or on a rotation.

**Website:** [earth-lens.netlify.app](https://earth-lens.netlify.app/)

![EarthLens](AppStore/screenshots/mac/01-earthlens-main.png)

## Requirements

macOS 26 Tahoe or later.

## Build

```bash
xcodebuild -project EarthLens.xcodeproj -scheme EarthLens -configuration Release build
```

Or open `EarthLens.xcodeproj` in Xcode.

## Privacy

No accounts, no analytics, no tracking. See [PRIVACY_POLICY.md](PRIVACY_POLICY.md).
